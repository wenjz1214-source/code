## 三、你还差哪一步才能拿到「真实性能指标」

1. **把 `trackformer_models_v1.zip` 弄到服务器**（任选其一）：
   - 在自己电脑/VPN 上 `wget` 官方链接，再 `scp` 到  
     `/share_data/wenjingzhong/graduation_project/models/`  
   - 或设置 `export TRACKFORMER_ZIP=/path/to/trackformer_models_v1.zip` 后执行  
     `bash scripts/download_models.sh`
2. 执行：
   ```bash
   source /share_data/wenjingzhong/graduation_project/venv_tf/bin/activate
   bash /share_data/wenjingzhong/graduation_project/scripts/check_env.sh
   bash /share_data/wenjingzhong/graduation_project/scripts/run_exp.sh
   ```
3. 在 `outputs/exp_*/a.log`、`b.log` 与 `docs/experiment_report.md` 中查看 **MOTA / IDF1** 等行。
